## Python: 10 Essential Features

This repository covers ten essential Python features.
Each feature has its own file so you can learn one topic at a time.

### Feature list

1. Syntax and indentation
2. Variables and data types
3. Control flow
4. Functions
5. Collections
6. Comprehensions
7. Modules and packages
8. Object-oriented programming
9. Error handling
10. File handling

### Repository purpose

- Provide short, focused notes for each Python feature.
- Keep examples separated by topic.
- Make it easy to review or practice one concept at a time.

### Suggested usage

- Start with the feature list above.
- Open the matching file for details and examples.
- Move through the files in order or jump to the topic you need.
